# 强哥备份工具

依据文件后缀名进行备份的 Windows 桌面工具。主要应用场景为使用某工程造价软件时被检测到锁文件的情况下，提供一份"随时可回退"的本地安全保障。

**绿色免安装，单文件运行。** 以系统托盘形式常驻后台，实时监测工程文件变更并自动创建加密备份。

---

## 下载与安装

1. 下载 `QiangBackup.exe`
2. 在任意磁盘新建目录（如 `D:\强哥备份`），将程序放置其中
3. 双击运行即可

---

## 首次使用

1. 点击**添加**按钮，选择需要备份的源文件夹（可添加多个）
2. 设置**备份存放目录**（建议与源文件夹不同磁盘，默认 `D:\强哥备份`）
3. 确认**文件扩展名**（默认已包含所有常见广联达工程文件类型，一般无需修改）
4. 点击**保存配置**
5. 点击**手动备份**完成首次备份
6. 点击**开始监控**，软件将最小化到系统托盘后台运行
7. 右键托盘图标 → 勾选**开机自启**

---

## 日常使用

### 后台监控模式（推荐）

点击"开始监控"后，软件常驻后台，文件保存后自动在几秒内完成备份。无需手动操作。关闭窗口后软件进入系统托盘，双击图标可重新打开。

### 手动备份

点击"手动备份" → 确认变更文件清单和预计耗时 → 开始备份。

### 一键恢复

点击"一键恢复" → 列表展示所有有备份版本的文件 → 选择文件和版本 → 恢复所选。

---

## 设置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 加密密码 | 7z 存档的解压密码 | `强哥备份` |
| 防抖时间 | 文件变更后等待多少秒再备份 | 3 秒 |
| 版本上限 | 每个文件最多保留几个备份版本 | 5 个 |
| 异常阈值 | 单次变更超过此数量自动停止监控 | 3 个 |
| 开机自启 | Windows 启动时自动运行 | 关闭 |

密码建议维持默认值"强哥备份"。加密的目的是**混淆**而非**防人**——防止工程造价软件在本地扫描磁盘时识别出备份文件。密码以明文存入配置文件。

---

## 文件移动与孤儿清理

### 文件移动

原文件移动位置后，软件会将其识别为新文件并重新备份。旧位置的备份存档会被标记为**孤儿文件**。

### 孤儿清理

点击"孤儿清理"按钮可查看并删除那些源文件已不存在的备份存档。删除孤儿文件时，如果所在文件夹内已无其他文件，文件夹也会自动向上递归删除。

---

## 完整性检查

点击"完整性检查"按钮可检测备份存档是否损坏（磁盘错误、意外断电等可能导致存档损坏）。检测到的损坏存档可选择性删除。

---

## 备份目录结构

```
源文件：  D:\项目\预算部\2025投标\报价总表.GBQ7
备份存档：D:\强哥备份\项目\预算部\2025投标\报价总表_20260525_143022_a1b2c3d4e5f6789.7z
                                 原始文件名      日期      时间     文件哈希
```

备份目录结构完全镜像源目录。每个源文件对应独立的加密 7z 存档，不打包合并。

---

## 配置文件

`config.json`（与程序同目录）存储所有配置。`backup.log` 存储操作日志。日志文件丢失不影响软件任何核心功能。

## 更换电脑或迁移数据

1. 将备份目录完整复制到新电脑
2. 运行程序，设置备份目录指向复制过来的路径
3. 密码必须和原来一致
4. 添加源文件夹后即可继续使用，所有历史备份自动关联
